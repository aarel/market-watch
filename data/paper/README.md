# Paper Universe Data

This directory contains data for the PAPER universe (broker-mediated paper trading).

Uses Alpaca paper trading endpoints with real market constraints.

## Files

- `config.json` - Paper-only runtime configuration
- `positions.json` - Current paper positions
- `equity.jsonl` - Paper equity curve (time series)

## Notes

- Results marked as PAPER_ONLY validity class
- Real market hours and rejections apply
- No real capital at risk
